URL Shortener Microservice

Input another website's URL into the address bar such as in the example below.

https://colin-fcc-url.herokuapp.com/https://google.com

If it's a valid URL, you'll get a JSON in the following format.

{
"original": "https://google.com",
"route": "https://colin-fcc-url.herokuapp.com/1"
}

Visiting the URL specified in the "route" value will redirect you to the original page.
